    - @login_required
    - No DB reads or writes

    - @login_required
    - Ownership check (user_id)
    - Active item check
    - No MTF scan

    - Returns {ok: true, item: ..., session_context: {...}}


================================================================================
5. WHERE latest_session_context IS STORED
================================================================================

  Column:  snapshot_json TEXT (LiveMonitorItem.snapshot_json)
  Keys:
    snap["latest_session_context"]    — full session context object
    snap["last_session_context_at"]   — ISO UTC timestamp of last update

  Written by:
    - POST /api/live-monitor/items/<id>/refresh       (before health compute)
    - POST /api/live-monitor/refresh-all              (before health compute)
    - POST /api/live-monitor/items/<id>/recalc-health (before health compute)
    - POST /api/live-monitor/items/<id>/refresh-session (session only, no health)

  Frontend reads from:
    item.snapshot.latest_session_context   (priority 1)
    item.snapshot.latest_health.session_context (priority 2)
    _lmGlobalSessionContext               (priority 3)
    UTC fallback label                    (priority 4)


================================================================================
6. HOW latest_health RECEIVES session_context
================================================================================

  In all 3 health refresh endpoints, after computing health:

    session_ctx = _lm_session_context()
    snap["latest_session_context"]  = session_ctx
    snap["last_session_context_at"] = session_ctx["computed_at"]

    health = _lm_compute_health(row, _snap=snap)
    health["session_context"] = session_ctx   ← attached AFTER compute
    snap["latest_health"] = health

  Important: session_context is added to the health dict AFTER
  _lm_compute_health returns. The health_score computation inside
  _lm_compute_health is NOT affected by session context at all.
  This phase is context-only. Score bonuses/penalties are Phase 5+.
