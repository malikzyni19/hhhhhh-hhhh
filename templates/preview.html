Scan engine
How the screener carves the chart into structure. Timeframe lives on the Main Scan controls so you can change it without opening this drawer.
Order Block
Demand / supply zones formed before the move that breaks structure. Selecting the OB module on the main page also activates these rules.
Fair Value Gap (FVG)
Inefficiency between three candles. These show up only when the FVG module is selected on the main page (or All).
Breaker Block
Failed OBs that flipped polarity. Selecting Breaker on the main page activates these.
Fibonacci
Retracement levels from the most recent swing leg. Selecting Fib on the main page activates these.
BTC Correlation
Optionally filter results by how a pair moves relative to BTC.
RSI thresholds
Levels used by the RSI alert and by the Momentum score in the Setup Insight panel.
TradingView chart link
Used by the View Chart button on every setup row.
